# إعداد Chatbot المتقدم

## المميزات الجديدة

✅ **الرد بالعربية افتراضياً** - الـ chatbot يرد بالعربية دائماً  
✅ **دعم OpenAI (اختياري)** - يمكن استخدام OpenAI GPT للردود الذكية  
✅ **بيانات جاهزة** - Seed data للأطباء والأقسام  
✅ **اقتراحات ذكية** - اقتراح الأطباء والعلاج والمواعيد  

## الإعداد الأساسي

الـ chatbot يعمل افتراضياً بدون أي إعداد إضافي ويرد بالعربية.

## تفعيل OpenAI (اختياري)

لتفعيل OpenAI للحصول على ردود أكثر ذكاءً:

### 1. الحصول على API Key من OpenAI
- اذهب إلى https://platform.openai.com/api-keys
- أنشئ حساب أو سجل الدخول
- أنشئ API Key جديد

### 2. تحديث appsettings.json

```json
{
  "ChatbotSettings": {
    "DefaultLanguage": "ar",
    "UseOpenAI": true,
    "OpenAIApiKey": "sk-your-api-key-here",
    "OpenAIModel": "gpt-3.5-turbo"
  }
}
```

### 3. إعادة تشغيل التطبيق

بعد تحديث الإعدادات، أعد تشغيل التطبيق.

## الإعدادات المتاحة

| الإعداد | الوصف | القيمة الافتراضية |
|---------|-------|-------------------|
| `DefaultLanguage` | اللغة الافتراضية | `ar` |
| `UseOpenAI` | تفعيل OpenAI | `false` |
| `OpenAIApiKey` | مفتاح API من OpenAI | `""` |
| `OpenAIModel` | نموذج OpenAI المستخدم | `gpt-3.5-turbo` |

## ملاحظات

- إذا لم يتم تفعيل OpenAI، سيستخدم الـ chatbot النظام القائم على القواعد (Rule-based)
- النظام القائم على القواعد يدعم العربية بشكل كامل
- OpenAI يعطي ردود أكثر ذكاءً ومرونة
- البيانات الجاهزة (Seed Data) متوفرة في `SeedData.cs`

## أمثلة الاستخدام

### بدون OpenAI (افتراضي)
```
المستخدم: "أعاني من ألم في القلب"
الـ chatbot: "بناءً على وصفك، أنصحك بالتواصل مع أحد الأطباء التاليين..."
```

### مع OpenAI
```
المستخدم: "أعاني من ألم في القلب"
الـ chatbot: [رد ذكي من OpenAI بالعربية مع اقتراحات مفصلة]
```

## البيانات الجاهزة

البيانات الجاهزة تشمل:
- ✅ أقسام المستشفى (Cardiology, Neurology, Orthopedics, إلخ)
- ✅ أطباء متعددين مع تخصصاتهم
- ✅ مواعيد ومرضى
- ✅ أدوية وفواتير

لتفعيل البيانات الجاهزة، تأكد من:
```json
{
  "SeedOptions": {
    "EnableSampleData": true
  }
}
```
