Sequence Diagram (Login Flow)

sequenceDiagram
    participant User
    participant Login
    participant AuthContext
    participant API
    participant Router

    User->>Login: Enter Email & Password
    User->>Login: Click Submit
    Login->>AuthContext: login(email, password)
    AuthContext->>API: POST /auth/login
    alt Success
        API-->>AuthContext: Return Token & User Data
        AuthContext-->>Login: Success
        Login->>Router: navigate('/')
    else Failure
        API-->>AuthContext: Error (401/400)
        AuthContext-->>Login: Throw Error
        Login->>User: Show Error Message
    end
